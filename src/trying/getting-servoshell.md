# Getting servoshell

The Servo project ships a small test browser called servoshell, which allows testing Servo.
servoshell is not a fully fledged browser, but is meant as a way to see how Servo works on your system and to run tests.
You can download the latest prebuilt version of servoshell from the [Downloads section](https://servo.org/download) on our website.
You can also download servoshell using the links below:
- [Windows Installer][windows-installer] ([sha256][windows-installer-sha256])
- [Windows ZIP][windows-zip] ([sha256][windows-zip-sha256])
- [macOS (x86)][macos-x86] ([sha256][macos-x86-sha256])
- [macOS (AArch64)][macos-aarch64] ([sha256][macos-aarch64-sha256])
- [Linux (x86)][linux-x86] ([sha256][linux-x86-sha256])
